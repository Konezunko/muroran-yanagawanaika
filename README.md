# 柳川内科医院 (Yanagawa Internal Medicine Clinic) - Demo Website

医療法人社団柳川内科医院の公式ウェブサイト用フロントエンドプロジェクト（デモ）です。
地域に根ざした「かかりつけ医」としての信頼感と、直感的に必要な情報へアクセスできる使いやすさを重視したデザインを実装しています。

## 🌐 ページ構成
本サイトは以下の5つのHTMLページで構成されています。
1. **ホーム (`index.html`)**: メインビジュアル、お知らせ、医院案内の概要、各種リンク
2. **ごあいさつ (`about.html`)**: 院長の挨拶や医院の理念
3. **はじめての方 (`first-visit.html`)**: 初診時の持ち物や受付から診察までの流れ
4. **診療案内 (`ambulatory.html`)**: 詳しい診療時間表、休診日、対応している診療分野
5. **アクセス (`access.html`)**: 医院の所在地、Googleマップ、駐車場、バスでのアクセス案内

## ✨ 特徴 (Features)
* **AIチャットボット**: 全てのページからワンクリック(🤖)で呼び出せるAIアシスタントを搭載。診療時間やアクセスに関するよくある質問に即座に答えます。
* **レスポンシブ対応**: スマートフォン、タブレット、PCなどあらゆるデバイスで見やすいレイアウト。
* **アクセシビリティ**: 必要な情報（アクセス、診療時間、電話番号など）へ最小限のクリックで辿り着けるUI設計。

## 🚀 ローカルでの動かし方 (How to run locally)
本プロジェクトはシンプルなHTML/CSS/JSで構成されているため、複雑なビルド（構築）環境は不要です。

1. このリポジトリをローカルにダウンロード・クローンします。
2. VSCodeの拡張機能「Live Server」を使用するか、Node.js環境がある場合はターミナルで以下のコマンドを実行してローカルサーバーを立ち上げます。
   ```bash
   npx live-server
   ```
3. ブラウザで `http://127.0.0.1:8080/` （またはターミナルに表示されたローカルURL）を開き、サイトを確認します。
